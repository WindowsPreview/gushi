屏蔽罩里面有一颗千兆交换机芯片，型号是RTL8211FS。主板边上有一颗NAND Flash闪存芯片，型号是F59L2G81A，容量256MB。好巨大的容量，相对TPLINK路由器来说。

取下两大块散热器，左边散热器与屏蔽罩接触用的是导热硅脂，右边用了几片硅胶垫片。

取下屏蔽罩：

先看CPU吧，型号是BCM4908，四核1.8GHz Cortex-A53 CPU。华硕AX88U、AX11000、网件RAX80、RAX200也是用它。

CPU旁边有两颗内存，型号是M15T4G16256A，单颗容量512MB，两颗组成1GB内存。

CPU外围有一颗看不见型号的芯片，我猜应该是2.5G网络芯片，型号是BCM54991E。博通方案的2.5G网口，估计都要使用它。

2.4G无线芯片型号是BCM43684，4x4mimo，2.4G 在40M频宽和802.11ax模式下最高速率1.2Gbps，手机因为2x2mimo，所以最高速率是573Mbps（小米10），如果手机只支持20MHz频宽，例如IQOO3，速率只有286Mbps。

无线芯片外围有四颗FEM芯片，上面丝印着4216，型号是QPF4216。The Qorvo® QPF4216 is an integrated front end module (FEM) designed for Wi-Fi 802.11ax systems.

集成了PA（功率放大器）和LNA（低噪放大器）。它的参数如下：

2400-2500 MHz
POUT = +22 dBm MCS11 HE40 -43 dB Dynamic EVM
POUT = +24 dBm MCS8/9 VHT40 -35 dB Dynamic EVM
POUT = +25 dBm MCS7 HT20/40 -30 dB Dynamic EVM
POUT = +26 dBm MCS0 HT20 Spectral Mask Compliance
Optimized for +5 V Operation
32 dB Tx Gain
1.8 dB Noise Figure
15.5 dB Rx Gain & 7 dB Bypass Loss
15 dB 5 GHz Rejection on Rx Path
Integrated Quasi-Log Power Detector

HE40就是WIFI6 40MHz频宽，MCS11就是1024-QAM调制方式，在这些状态下，输出功率+22 dBm=160mW，理论上想让2.4G信号很强，可以设置在64-QAM和20M频宽，这样就有+25dbm=320mW。数据看着是这样。有兴趣的你们可以试试。

5G无线芯片型号也是BCM43684，4x4mmo，支持160MHz频宽，最高速率4.8Gbps，只支持80MHz频宽的小米10，连接速率是1.2Gbps。

外围有四颗FEM芯片，上面丝印着4516，型号是QPF4516，但找不到详细资料，以后找到再作补充吧。

至此，拆机结束。

拿华硕AX6600做个简单的speedtest测速。AX6600是三频，有一颗5G芯片是它的CPU BCM6755集成的，它只能工作在低信道，而另一个5G芯片BCM43684，两台都一样，有可比性，但只能用在高信道。2.4G信道都是9，5G低信道44，5G高信道161。（华硕AX6600设置美国地区，TP默认打开了信号增强）

USB3.0速度也简单测试了，TP的XDR6060在2.5G口下读取USB3.0，读取速度达到148MB/秒，写入速度117MB/秒。可以远程访问共享。以后再做详细的测试，本篇只是拆机文文章。
